# AllerLens

**An Allergen Detector for Ingredient Lists Using the Boyer-Moore String Matching Algorithm**

## 🚀 Objectives

- Extract text from food ingredient labels using Tesseract OCR.
- Utilize a Hash Map for O(1) retrieval of allergen aliases from a static JSON library.
- Implement the **Boyer-Moore Algorithm** to efficiently scan extracted text from hidden allergens.
- Implement **Brute Force String Matching** as a baseline to benchmark algorithmic execution times.

## 💻 Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Python, Flask/FastAPI
- **Core Engine:**: Tesseract OCR

## 📁 Repo Structure

- `\frontend` - Contains the React user interface and components.
- `\backend` - Contains the Python server, OCR processing, and algorithm implementations.
- `\docs` - Contains project documentation and time complexity reports.
